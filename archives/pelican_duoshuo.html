Pelican 中使用多说评论系统

Date 2015-05-30 15:54 (星期六) Tags Pelican / 多说

折腾 Pelican 两天了，感觉很好用，很容易上手。照着教程使用了 disqus 的评论系统， 超简单的在配置文件 pelicanconf.py 中加入 DISQUS_SITENAME = shortname 就可以了。看起来也不错， 因为刚开始用没人给我评论过。但是问题也有。

  1. 用的是国外的社交账号。伟大的墙的关系，那些网站在中国是不存在的啊
  2. 首页打开的时候,加载了一个 disqus_scripts.html 的页面，那个页面会加载一个统计的count.js。然后 大家都知道的伟大的墙的存在，其中使用到了google的统计代码，所以会导致完全打开页面会很慢很慢。 当然也可以在模板的 base.html 中将 include 这个页面这句的代码删除已解决这个问题。
  3. 打开页面加载评论系统的速度也不稳定，甚至加载失败的问题也有。

所以果断弃之改用国内的多说来替换掉这个系统。步骤也很简单两分钟搞定。

  1. 上多说点击我要安装随便用个国内的社交账号登陆上去。然后再填点信息，最后会得到一段通用的代码

    <!-- 多说评论框 start -->
    <div class="ds-thread" data-thread-key="请将此处替换成文章在你的站点中的ID" data-title="请替换成文章的标题" data-url="请替换成文章的网址"></div>
<!-- 多说评论框 end -->
<!-- 多说公共JS代码 start (一个网页只需插入一次) -->
<script type="text/javascript">
var duoshuoQuery = {short_name:"多说用户名"};
    (function() {
        var ds = document.createElement('script');
        ds.type = 'text/javascript';ds.async = true;
        ds.src = (document.location.protocol == 'https:' ? 'https:' : 'http:') + '//static.duoshuo.com/embed.js';
        ds.charset = 'UTF-8';
        (document.getElementsByTagName('head')[0] 
         || document.getElementsByTagName('body')[0]).appendChild(ds);
    })();
    </script>
<!-- 多说公共JS代码 end -->

  2. 配置文件中如 disqus 一样加入一句 DUOSHUO_SITENAME = "多说用户名" （必须全部大写，否则pelican不认。）

  3. 进入模板的 includes 文件夹，新建一个文件，如：“duoshuo.html”。把上面的内容复制进去。
    1. 将替换里的那几句分别替为 {{ article.slug }} data-title="{{ article.title }} data-url="{{ SITEURL }}/{{ article.url }}"
    2. 首尾分别加入 {% if DUOSHUO_SITENAME %} 和 {% endif %}
    3. 将多说的用户名改成 {short_name: '{{ DUOSHUO_SITENAME }}'}; （可选）
  4. 回到模板文件夹找到 article.html 文件，在很多 include 的地方加入 {% include 'includes/duoshuo.html' %} 这句。
  5. 完工。

最后的 duoshuo.html 文件里面像这样：

{% if DUOSHUO_SITENAME %}
<!-- 多说评论框 start -->
    <div class="ds-thread" data-thread-key="{{ article.slug }}" data-title="{{ article.title }}" data-url="{{ SITEURL }}/{{ article.url }}"></div>
<!-- 多说评论框 end -->
<!-- 多说公共JS代码 start (一个网页只需插入一次) -->
    <script type="text/javascript">
    var duoshuoQuery = {short_name: '{{ DUOSHUO_SITENAME }}'};
    (function() {
        var ds = document.createElement('script');
        ds.type = 'text/javascript';
        ds.async = true;
        ds.src = (document.location.protocol == 'https:' ? 'https:' : 'http:') + '//static.duoshuo.com/embed.js';
        ds.charset = 'UTF-8';
        (document.getElementsByTagName('head')[0] 
         || document.getElementsByTagName('body')[0]).appendChild(ds);
    })();
    </script>
<!-- 多说公共JS代码 end -->
{% endif %}
