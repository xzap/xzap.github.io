pelican配置文件

Date 2015-05-26 16:11 (星期二) Tags Pelican

随着博客浪潮的过去,现在写博客也变得不怎么流行了,我的博客也是几度更替.总是被停止服务.github却逆向开通了github page.不过虽然早就想试试这个服务,但是好像很难的样子,因为只支持静态啊.现在有了pelican用起来还是很爽的. 1. 支持语法高亮, 2. 支持markdown 3. 很方便的可以用github管理 4. 很爽的disqus的评论系统接入,只要打个disqus用户名就可以 5. ……

下面是我七拼八凑的配置文件: :::python3 #!/usr/bin/env python # -- coding: utf-8 -- # from future import unicode_literals 

AUTHOR = 'xzap'
SITENAME = "xzap's 缓存区"
SITEURL = 'http://xzap.github.io'
# SITEURL = ''
PATH = 'content'
PAGE_PATHS = ['pages']
ARTICLE_PATHS = ['articles']
STATIC_PATHS = ['images', 'files']
DEFAULT_DATE_FORMAT = ('%Y-%m-%d %H:%M:%S (%A)')
DATE_FORMAT={"zh":("zh_CN","%Y-%m-%d,%X,%A"),}#日期格式设置，可按自己喜好设定

EXTRA_PATH_METADATA = {
  'files/robots.txt': {'path': 'robots.txt'},
  'images/favicon.ico': {'path': 'favicon.ico'},
}
TIMEZONE = 'Asia/Shanghai'

DEFAULT_LANG = 'zh'

# Feed generation is usually not desired when developing
# FEED_ALL_ATOM = None
# CATEGORY_FEED_ATOM = None
# TRANSLATION_FEED_ATOM = None
# AUTHOR_FEED_ATOM = None
# AUTHOR_FEED_RSS = None

# Blogroll
LINKS = (('Pelican', 'http://getpelican.com/'),
         ('Python.org', 'http://python.org/'),
         ('Jinja2', 'http://jinja.pocoo.org/'),
         )

# Social widget
SOCIAL = (('You can add links in your config file', '#'),
          ('Another social link', '#'),)

DEFAULT_PAGINATION = 10


# Theme
THEME = 'pelican-themes/pelican-bootstrap3'
COVER_BG_COLOR = '#375152'
DEFAULT_PAGINATION = 10

# Plugin
PLUGIN_PATHS = ["pelican-plugins"]
PLUGINS = [ 'sitemap', 'gravatar', 'summary', 'neighbors', 'related_posts',  ]

# Sitemap
SITEMAP = {
  'format': 'xml',
  'priorities': {
    'articles': 1,
    'pages': 0.9,
    'indexes': 0.8,
  },
  'changefreqs': {
    'indexes': 'daily',
    'articles': 'daily',
    'pages': 'weekly'
  }
}
GITHUB_URL = "http://xzap.github.io"

DISQUS_SITENAME = "xzap"

FEED_ALL_ATOM = 'feeds/all.atom.xml'
CATEGORY_FEED_ATOM = 'feeds/%s.atom.xml'

# Uncomment following line if you want document-relative URLs when developing
#RELATIVE_URLS = True
ARTICLE_URL = 'archives/{slug}.html'
ARTICLE_SAVE_AS = 'archives/{slug}.html'

YEAR_ARCHIVE_SAVE_AS = 'archives/{date:%Y}/index.html'
MONTH_ARCHIVE_SAVE_AS = 'archives/{date:%Y}/{date:%m}/index.html'
MENUITEMS_AFTER = (('RSS', '/feeds/all.atom.xml'),)
# MENUITEMS = (('RSS', '/feeds/all.atom.xml'),)

CACHE_CONTENT = True
LOAD_CONTENT_CACHE = True
CHECK_MODIFIED_METHOD = 'mtime'
